A simple calculator made with Python and PySide6. Its able to do the 4 basic aritmethic operacions as well as potentiation and root extraction.
To avoid floating point issues, every number is first entered as a string and then converted to float after typing is complete and a new operation is called.
Comments in the code file are written in portuguese for Brazilian users, but there will be comments in English added in the future as I add new functionalities.
Feel free to comment, fork, download, propose changes and improvements.

Changelog:

0.1  Values can be entered via keyboard as well as by clicking on the respective buttons. Users can type in values, delete every digit, enabling and disbaling decimals input as well as clearing all values in memory.
To disable decimals input, the user is required to delete every decimal digit, perform an arithmethic operation or clear all values.
